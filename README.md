# 商品データ収集ツール（デモ）

Webサイトの商品一覧から **商品名・価格・評価・在庫状況** を自動収集し、
Excelで開けるCSVファイルに整えて出力するツールです。

## できること

- 商品一覧ページを**自動でページ送り**しながら全件収集
- 価格は記号を除いた**数値**に整形（そのまま集計・グラフ化できる）
- 星評価を数値（1〜5）に変換
- 在庫状況を「在庫あり／なし」に整理
- 出力は**Excelで文字化けしないCSV**（UTF-8 BOM付き）
- 相手サーバーに負荷をかけない**待機時間**を内蔵

## 実行例

```bash
python scraper.py            # 全ページ収集
python scraper.py --pages 3  # 最初の3ページだけ
```

## 出力サンプル（products.csv）

| 商品名 | 価格 | 評価(星) | 在庫 |
|---|---|---|---|
| A Light in the Attic | 51.77 | 3 | 在庫あり |
| Sapiens: A Brief History of Humankind | 54.23 | 5 | 在庫あり |
| The Requiem Red | 22.65 | 1 | 在庫あり |

→ 1回の実行で **60件以上を数秒で収集**。

## 応用

このデモは練習用の公開サイト（books.toscrape.com）を対象にしていますが、
抽出ロジックを対象サイトに合わせて調整することで、
**価格調査・競合分析・在庫モニタリング・市場リサーチ** などに応用できます。

- 「毎日決まった時間に自動収集」する定期実行
- 「前日との価格差を検知して通知」する仕組み
- 収集データをグラフ・レポート化

といったカスタマイズも承れます。お気軽にご相談ください。
